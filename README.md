# PROJECT-1-Car-Price-Prediction-ML
A machine learning regression project for predicting car prices using Python, exploratory data analysis, feature engineering, and scikit-learn.
Car Price Prediction Using Machine Learning


<h1>Car Price Prediction Using Machine Learning</h1>

<h2>Project Overview</h2>
<p>
This project focuses on predicting car prices using regression-based Machine Learning techniques.
The entire workflow—from data preprocessing and exploratory data analysis (EDA) to model training
and evaluation—was implemented using Python in a Kaggle Notebook environment.
</p>
<p>
The objective of this project is to gain hands-on experience with real-world datasets and apply
regression models to solve price prediction problems effectively.
</p>

<h2>Model Performance</h2>
<ul>
    <li><strong>R² Score:</strong> Approximately 83%</li>
</ul>

<h3>Evaluation Metrics</h3>
<ul>
    <li>R² Score</li>
    <li>Adjusted R² Score</li>
</ul>
